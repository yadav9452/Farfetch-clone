## Run Locally

Clone the project

```bash
  git clone https://github.com/Akshay-Singh-Rajput/Farfetch-clone.git
```

Go to the project directory

```bash
  cd Farfetch-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
